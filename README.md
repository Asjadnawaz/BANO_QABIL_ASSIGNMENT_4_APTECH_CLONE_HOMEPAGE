# Aptech Learning Pakistan — Clone Homepage

A pixel-faithful front-end clone of the [Aptech Learning Pakistan](https://aptech-education.com.pk) website, built as **Assignment 4** for the **Bano Qabil Web 1 Course**.

---

## 📌 Project Overview

This project replicates the look and layout of the official Aptech Learning Pakistan homepage, including its navigation, hero section, course banners, global statistics, testimonials, and footer — using only HTML and CSS.

---

## 🖥️ Pages

| File | Description |
|------|-------------|
| `index.html` | Main homepage clone |
| `style.css` | Stylesheet for the homepage |
| `courses.html` | Courses listing page |
| `courses.css` | Stylesheet for the courses page |

---

## ✨ Features

- **Responsive Navbar** — Logo, navigation links, and a search icon
- **Hero Section** with:
  - Career guidance sidebar form (name, mobile number)
  - Most-searched topics list
  - Contact / WhatsApp info
  - Auto-sliding banner carousel (3 slides)
  - Career course cards for Matric & Intermediate students
- **Program Banners** — ACN Pro, Smart Pro, Quick Skills, Communication Excellence, Future Ready Careers
- **Global Stats Section** — 5 Continents · 40 Countries · 2500 Centers
- **Embedded YouTube Video** — ACCP-AI curriculum walkthrough
- **Testimonials Carousel** — student/alumni testimonials
- **Detailed Footer** — initiatives, honors & awards, social links, affiliations (SBTE, STEVTA), and bottom legal links

---

## 🛠️ Tech Stack

- **HTML5** — semantic markup
- **CSS3** — custom styling, flexbox layout, carousel animations
- **Font Awesome 6.5** — icons (CDN)
